# DSA-C Weekly Lab Repository

This repository collects weekly C programming and data structures exercises completed for the ALU DSA lab. It brings together the source code, reflective notes, and visual aids used to study how common data structures and algorithms work in practice.

## Overview

The repository is organized by week and session. Each session focuses on one topic, with a mix of implementation files, supporting notes, and analysis documents.

| Folder | Focus |
| --- | --- |
| `1st-week/1st-session/` | Array operations, CRUD logic, and search complexity analysis |
| `1st-week/2nd-session/` | Singly linked lists, recursion notes, and pointer visualization |
| `2nd-week/` | Tree traversal and binary search tree practice |
| `3rd-week/` | Graph adjacency representation and depth-first search |
| `4th-week/` | Heap structure exploration and Huffman coding |
| `5th-week/` | Shortest path algorithms such as Dijkstra and Bellman-Ford |
| `6th-week/` | N-ary tree DFS and related operations |

## What You Will Find

- Clean C implementations for each weekly topic.
- Source code snapshots that show how the solution evolved.
- Reflection and analysis files that explain complexity, design choices, and learning outcomes.
- Executable files in some sessions for quick testing on Linux systems.

## How To Use The Repository

1. Open the week and session folder you want to study.
2. Read the `README.md` file in that folder first.
3. Review the source file and the related analysis notes.
4. Compile the program with `gcc` when a runnable C file is provided.

Example:

```bash
gcc array_operations.c -o array_operations
./array_operations
```

## Learning Goals

This repository is meant to support hands-on practice with:

- Arrays and searching
- Linked lists and dynamic memory
- Trees and traversal techniques
- Graph representations and DFS
- Heap operations and Huffman coding
- Shortest path algorithms

Each folder is intentionally small and focused so the implementation and the theory can be studied together.
